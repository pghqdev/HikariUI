Buttons

Dialog

Native <dialog>, opened as a modal. Open/close ride the native command invoker — no JS unless the browser lacks it.

Ship it?

A modal on bare <dialog>. Scrim is a flat neutral dim; the box enters and leaves on --ease-hikarion. Press Esc or a button to close.

Tabs

Bare buttons become an underline tablist. Arrow keys move between tabs; the selected edge is the one rationed accent.

Accordion

What is Hikarion?

Bare <details>/<summary>. The chevron rotates on --ease-hikarion; the toggle is native, no JS.

Does it need JavaScript?

No — disclosure is native. hikarion.js is only for the few things CSS can't do.

What about reduced motion?

The chevron transition is disabled by the global prefers-reduced-motion killswitch.
